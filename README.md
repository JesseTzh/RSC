## 💡 简介

[RSC](https://github.com/JesseTzh/RSC) Remote Shell Commander 的缩写，旨在通过 Web 接口直接执行 Shell 脚本或命令。

适用于不方便 SSH 访问的情景中远程执行脚本或 Shell 命令，甚至可以用来做一些简单运维工作。

>⚠️警告：由于本程序本身不具有鉴权功能，请在使用时注意安全问题。
